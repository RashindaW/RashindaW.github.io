# Rashinda — Personal Site (GitHub Pages)

A no-build, static website ready to publish on GitHub Pages.

## Quick Publish

1) Create a repo named `<your-username>.github.io` on GitHub.
2) Download this folder as a ZIP, unzip it, and copy contents into the repo.
3) Commit & push. GitHub Pages serves it automatically at `https://<your-username>.github.io/`.

## Update Your Info

- **CV**: Replace `assets/cv/Rashinda.pdf` with your latest CV (keep the same filename or update the links).
- **Photos**: Replace files in `assets/img/` with your project/medal photos.
- **Homepage text**: Edit `index.html` intro and featured works.
- **Projects**: Edit `projects.html` cards, add new ones by copying a card block.
- **Books**: Edit `books.html` with your summaries.
- **Philosophy**: Edit `philosophy.html` for your ideas/notes.

## Optional: Custom Domain
- Create a `CNAME` file in the repo root with your domain name (e.g., `rashinda.dev`).
- Add an `A` or `CNAME` record at your DNS provider per GitHub Pages docs.

## Accessibility & SEO
- Update `<meta name="description">` in each page.
- Add descriptive `alt` text on images.
- Keep link text meaningful (avoid “click here”).

## Local Preview
- Just open `index.html` in your browser. No build tools required.
